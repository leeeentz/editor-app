A simple vector graphics editor built with React and p5.js, allowing users to create and manipulate 2D shapes.

- Canvas-based rendering using p5.js
- Shape management with polygon objects
- Interactive toolbar for switching between editing modes
- Real-time transformations (rotation, scaling, translation)
- Modular architecture

# Clone the repository
git clone https://github.com/leeeentz/editor-app

# Install dependencies
npm install

# Start development server
npm start
